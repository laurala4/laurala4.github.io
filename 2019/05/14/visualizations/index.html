The importance of visualization

In the set of videos about “The Information Visualization Mantra” by University of Illiois we learned about Schneiderman’s Mantra. 1. Overview First 2. Zoom and Filter 3. Then Detail on Demand

It has been said, “a picture is worth a thousand words”. Visualization is able to provide a better understanding of the data being reviewed, with a graph, color and motion.

Today’s computers are able to process large amounts of data in a small amount of time. This technology has made it possible to review, process algorithms and display information as maps, graphs, pictures, and videos. This way we can communicate effectively using color, pictures, videos, graphs.

Magnifying data using zoom, focus & context, distortion lens is a fantastic way to focus on data as you zoom in on the part that you are interested in.

Filtering allows us to remove extraneous data providing more room for detail that we are interested in to show details on demand.
